## Overview

Perplexity Professional Suite is a complete AI research platform with file analysis, collections, and advanced models for professional research workflows.

File analysis processes uploaded documents, PDFs, and images within research conversations, enabling questions about specific documents and integration of personal materials with web research.

Collections organize research threads by topic or project, building a persistent research library that accumulates knowledge across sessions.

Pro Search uses advanced reasoning with step-by-step research planning that handles complex, multi-part questions requiring synthesis across many sources.

Perplexity Professional Suite is the AI research platform for professionals who need comprehensive, sourced research with the file handling and organizational tools that professional knowledge work demands.

## Performance & Integration

Perplexity is engineered around macOS technologies that handle performance-critical work efficiently. Tasks run in the background without slowing down the rest of your system — you can keep working while Perplexity processes operations quietly behind the scenes.

The app integrates deeply with the macOS ecosystem out of the box:

- Launch workflows directly through **Spotlight**
- Preview files instantly with **Quick Look**
- Trigger actions from any app via the **Services menu**
- Fully compatible with **Shortcuts** for automated workflows
